Политика конфиденциальности

AdShorts AI

Дата последнего обновления: 2 марта 2026 г.

Настоящая Политика конфиденциальности описывает порядок сбора, использования и защиты информации пользователей сервиса AdShorts AI (далее — «Сервис»).

Используя Сервис, Пользователь выражает согласие с настоящей Политикой конфиденциальности.

1. Какие данные мы собираем

Сервис может обрабатывать следующие данные:

  • техническую информацию (IP-адрес, тип устройства, браузер);
  • идентификаторы аккаунта (например, Telegram user ID);
  • данные, передаваемые Пользователем при использовании Сервиса;
  • информацию о действиях в Сервисе (лог-файлы, события).

Сервис не требует предоставления паспортных, платёжных или иных чувствительных персональных данных, за исключением случаев, предусмотренных функциональностью платёжных систем.

2. Цели обработки данных

Данные обрабатываются исключительно для:

  • предоставления и улучшения работы Сервиса;
  • обеспечения безопасности и стабильности;
  • обработки обращений Пользователей;
  • выполнения требований законодательства.

3. Хранение и защита данных

3.1. Сервис принимает разумные технические и организационные меры для защиты информации от утраты, несанкционированного доступа или раскрытия.

3.2. Данные хранятся в течение срока, необходимого для достижения целей обработки, либо в сроки, предусмотренные законодательством.

4. Передача данных третьим лицам

Сервис может передавать данные третьим лицам только в следующих случаях:

  • платёжным провайдерам — для обработки платежей;
  • хостинг-провайдерам и техническим подрядчикам — для обеспечения работы Сервиса;
  • государственным органам — в случаях, предусмотренных законом.

Сервис не продаёт и не передаёт персональные данные Пользователей в рекламных целях.

5. Cookies и аналитика

Сервис может использовать файлы cookie и аналогичные технологии для:

  • корректной работы интерфейса;
  • аналитики и улучшения пользовательского опыта.

Пользователь может ограничить использование cookie в настройках браузера.

6. Права Пользователя

Пользователь имеет право:

  • запрашивать информацию о своих данных;
  • требовать уточнения или удаления данных, если это не противоречит закону;
  • прекратить использование Сервиса в любой момент.

7. Данные YouTube и Google API Services

Если Пользователь подключает YouTube-канал через Google OAuth, AdShorts AI запрашивает доступ только к конкретным разрешениям (scopes), указанным на экране согласия Google.

7.1. Scope: https://www.googleapis.com/auth/youtube.upload

Используется исключительно для:

  • загрузки видео на YouTube-канал Пользователя;
  • установки метаданных видео при загрузке (название, описание, теги);
  • публикации видео по прямому запросу Пользователя.

Данный scope не предоставляет доступа к Gmail, Google Диску, контактам или другим сервисам Google.

7.2. Scope: https://www.googleapis.com/auth/youtube.readonly

Используется только для:

  • получения названия YouTube-канала Пользователя;
  • получения идентификатора (ID) YouTube-канала;
  • получения аватара (обложки) канала;
  • отображения выбранного канала в интерфейсе бота;
  • подтверждения правильного канала для публикации.

Мы не изменяем, не удаляем и не управляем существующим контентом YouTube через доступ только для чтения.

7.3. Использование и хранение данных

  • OAuth-токены доступа хранятся в зашифрованном виде.
  • Токены используются только для выполнения авторизованных действий.
  • Пользователь может отозвать доступ в любой момент через настройки аккаунта Google.
  • При отключении канала хранимые токены удаляются безвозвратно.
  • Данные контента YouTube не хранятся постоянно — только в объёме, строго необходимом для обеспечения функции публикации.
  • Название и ID канала могут храниться только для обеспечения функции публикации и удаляются по запросу пользователя.

7.4. Соответствие требованиям Limited Use

Использование информации, полученной от Google API Services, в AdShorts AI соответствует Google API Services User Data Policy, включая требования об ограниченном использовании (Limited Use).

Данные пользователей Google:

  • не продаются;
  • не передаются третьим лицам;
  • не используются в рекламных целях;
  • не используются для обучения моделей искусственного интеллекта или машинного обучения.

Подключая YouTube через Сервис, Пользователь также соглашается с Условиями использования YouTube и Политикой конфиденциальности Google: https://policies.google.com/privacy.

8. Отзыв доступа и удаление данных

Пользователь может:

  • отозвать доступ к Google-аккаунту в настройках разрешений: myaccount.google.com/permissions;
  • запросить удаление данных через страницу Удаление данных или по адресу support@adshortsai.com.

9. Изменения Политики

Сервис вправе изменять настоящую Политику конфиденциальности без предварительного уведомления.
Актуальная версия всегда доступна на сайте adshortsai.com.
