Privacy Policy

AdShorts AI

Last updated: March 2, 2026

This Privacy Policy describes the procedures for collecting, using, and protecting information of users of the AdShorts AI service (hereinafter referred to as the "Service").

By using the Service, the User agrees to this Privacy Policy.

1. What Data We Collect

The Service may process the following data:

  • technical information (IP address, device type, browser);
  • account identifiers (e.g., Telegram user ID);
  • data provided by the User when using the Service;
  • information about actions in the Service (log files, events).

The Service does not require passport, payment, or other sensitive personal data, except in cases provided by payment system functionality.

2. Purposes of Data Processing

Data is processed exclusively for:

  • providing and improving the Service;
  • ensuring security and stability;
  • processing User requests;
  • complying with legal requirements.

3. Data Storage and Protection

3.1. The Service takes reasonable technical and organizational measures to protect information from loss, unauthorized access, or disclosure.

3.2. Data is stored for the period necessary to achieve processing purposes, or for periods provided by law.

4. Disclosure to Third Parties

The Service may transfer data to third parties only in the following cases:

  • to payment providers — for payment processing;
  • to hosting providers and technical contractors — to ensure Service operation;
  • to government authorities — in cases provided by law.

The Service does not sell or transfer User personal data for advertising purposes.

5. Cookies and Analytics

The Service may use cookies and similar technologies for:

  • proper interface operation;
  • analytics and user experience improvement.

The User may limit cookie usage in browser settings.

6. User Rights

The User has the right to:

  • request information about their data;
  • request correction or deletion of data, if not contrary to law;
  • stop using the Service at any time.

7. Google User Data

7.1. Data Accessed

AdShorts AI accesses the following Google user data via YouTube Data API:

  • YouTube channel basic information (channel name, channel ID);
  • permission to upload videos;
  • permission to set video metadata during upload (title, description, tags);
  • permission to publish videos as part of the upload flow.

We do NOT access:

  • Gmail data;
  • Google Drive data;
  • Google Contacts;
  • any unrelated Google services.

7.2. How We Use Google Data

Google user data is used strictly for:

  • uploading and publishing videos to the user's connected YouTube channel;
  • setting video metadata during upload (title, description, tags);
  • displaying upload status inside the Telegram bot.

We do NOT:

  • sell Google user data;
  • transfer Google user data to third parties;
  • use Google user data for advertising purposes;
  • use Google user data to train artificial intelligence or machine learning models.

7.3. Data Storage

  • OAuth tokens are stored securely in encrypted format;
  • tokens are used only to perform authorized actions;
  • users may revoke access at any time via Google Security Settings;
  • upon user request, all stored tokens are deleted.

7.4. Data Retention

  • access tokens are stored only while the user keeps the channel connected;
  • if the user disconnects YouTube, tokens are permanently deleted.

7.5. Compliance

AdShorts AI complies with:

  • Google API Services User Data Policy, including Limited Use requirements;
  • YouTube API Services Terms of Service.

By connecting YouTube through the Service, the User also agrees to the Google Privacy Policy.

8. Access Revocation and Data Deletion

The User may:

  • revoke Google account access in permissions settings: myaccount.google.com/permissions;
  • request data deletion via Data Deletion or by email at support@adshortsai.com.

9. Policy Changes

The Service reserves the right to change this Privacy Policy without prior notice.
The current version is always available at adshortsai.com.

10. Contact Information

For data processing questions, the User may contact:
