Удаление данных пользователей

Дата последнего обновления: 23 февраля 2026 г.

Пользователь может запросить удаление своих данных, связанных с использованием приложения AdShorts AI.

Для удаления данных необходимо отправить запрос на электронный адрес: support@adshortsai.com

В письме укажите:

  • адрес электронной почты, связанный с аккаунтом (если есть);
  • Telegram user ID и/или YouTube channel ID (если подключали YouTube);
  • тему письма: Data Deletion Request — AdShorts AI.

Срок обработки запроса на удаление — до 30 календарных дней, если более длительное хранение не требуется по закону.

При подключении YouTube через Google OAuth вы также можете отозвать доступ приложения в Google: myaccount.google.com/permissions.

Подробные условия обработки данных: Политика конфиденциальности.
